# To-Do App in React for learning purpose

### Clone and run:
    - yarn install 
    - yarn dev
   

### Components:

    - New Todo Form
    - Todo List
    - Todo Item

### States:

    - todos
    - newItem

## Functions:

    - addTodo (App.jsx)
    - toggleTodo (App.jsx)
    - deleteTodo (App.jsx)
    - handleSubmit (NewTodoForm.jsx)

## Storage:

    - localStorage
